# SQLServer
Implementing basic SQL Server operations from scratch (Create, Delete, Select, Update, Drop) 

## Data types in our SQL Server:
- String
- Integer
- double
- boolean
- DateTime
- Date
- Time

## Operations our SQL Server can perform:
- CREATE DATABASE 
- CREATE TABLE
- INSERT
- SELECT
- DROP
- DELETE
